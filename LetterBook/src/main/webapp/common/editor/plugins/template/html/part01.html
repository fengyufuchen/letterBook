章节小标题：如 1、初始化操作

第1步 需求分析

问题描述：假设单链表L不存在，初始化一个带头结点的空链表L。
初始条件：单链表L不存在。
操作结果：创建一个带头结点的空链表,L 为指向头结点的指针，头结点分配成功返回OK。
为头结点分配空间，空间分配失败则报错  
函数头：Status InitList_L(LinkList &L)，L是引用参数。
问题分析：
  1. 完成此操作，要掌握C语言中分配内存的函数malloc，要注意这个函数的参数和返回值。
  2. malloc分配的空间类型是单链表的结点类型，由于该结点是头结点且是目前单链表中的唯一结点，因此要注意正确设置结点的指针域。
asdfasdfsafsdf
例外情况：头结点分配失败返回ERROR。
